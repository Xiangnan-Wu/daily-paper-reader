## 摘要
将视觉-语言-动作（VLA）模型集成到自动驾驶系统中，为解释复杂场景和执行控制指令提供了一个统一的框架。然而，为了实现准确的时间推理，必须纳入历史多视图帧，这带来了沉重的计算负担，主要源于大语言模型（LLM）中自注意力机制的二次方复杂度。为了缓解这一瓶颈，我们提出了 ETA-VLA，一种用于 VLA 模型的高效 Token 适配框架。ETA-VLA 处理过去 $n$ 帧的多视图图像，并引入了一种新颖的 LLM 内部稀疏聚合器（ILSA）。受人类驾驶员注意力分配的启发，ILSA 在文本查询和时间一致性的引导下，动态地识别并剪枝冗余的视觉 Token。具体而言，我们利用文本引导的评分机制以及保持多样性的稀疏化策略来选择关键 Token 的稀疏子集，从而确保对驾驶场景的全面感知。在 NAVSIM v2 上的广泛实验表明，ETA-VLA 实现了与最先进基准模型相当的驾驶性能，同时将计算 FLOPs 降低了约 32%。值得注意的是，我们的方法剪枝了 85% 的视觉 Token，并将推理 FLOPs 降低了 61%，但在 NAVSIM v2 基准测试中仍保留了 94% 的原始准确率。

## Abstract
The integration of Vision-Language-Action (VLA) models into autonomous driving systems offers a unified framework for interpreting complex scenes and executing control commands. However, the necessity to incorporate historical multi-view frames for accurate temporal reasoning imposes a severe computational burden, primarily driven by the quadratic complexity of self-attention mechanisms in Large Language Models (LLMs). To alleviate this bottleneck, we propose ETA-VLA, an Efficient Token Adaptation framework for VLA models. ETA-VLA processes the past $n$ frames of multi-view images and introduces a novel Intra-LLM Sparse Aggregator (ILSA). Drawing inspiration from human driver attention allocation, ILSA dynamically identifies and prunes redundant visual tokens guided by textual queries and temporal consistency. Specifically, we utilize a text-guided scoring mechanism alongside a diversity-preserving sparsification strategy to select a sparse subset of critical tokens, ensuring comprehensive awareness of the driving scene. Extensive experiments on the NAVSIM v2 demonstrate that ETA-VLA achieves driving performance comparable to state-of-the-art baselines while reducing computational FLOPs by approximately 32\%. Notably, our method prunes 85% of visual tokens and reduces inference FLOPs by 61\%, but still retaining 94% of the original accuracy on the NAVSIM v2 benchmark.

#### 1. 核心问题与研究动机

*   **研究动机**：借鉴人类驾驶员的注意力分配机制——即在驾驶时会动态关注关键区域（如前方道路、行人）而忽略冗余背景，同时保留对盲区的基本感知。论文旨在通过高效的 Token 适配技术，在不损失精度的前提下大幅降低计算量。

#### 2. 方法论：核心思想与技术细节
ETA-VLA 框架由两个互补模块组成：
*   **时间融合模块 (TFM)**：
    *   在进入 LLM 之前，利用可学习的时间嵌入（Time Embedding）为多帧视觉特征注入时序信息。
    *   通过 Transformer 编码器捕捉时间依赖，并使用**可学习的时间权重**进行加权聚合，将多帧特征压缩为单帧维度的紧凑表示，从源头上缓解 Token 增长。
*   **LLM 内部稀疏聚合器 (ILSA)**：
    *   **动态文本锚点选择**：根据视觉特征与指令的相似度，动态筛选出最相关的文本 Token 作为“评分者”，避免无关指令噪声干扰。
    *   **无 RoPE 语义评分**：在计算注意力权重时移除旋转位置编码（RoPE）的影响，消除因 Token 距离产生的偏置，确保评分纯粹基于语义相关性。
    *   **多样性保留稀疏化策略**：采用两阶段筛选。首先进行**全局语义剪枝**（保留最相关的区域）；其次进行**多样性回收**（在被剪掉的 Token 中按视图回收特征差异大的 Token），确保模型不会因过度关注局部而丢失对侧方或后方关键空间（如盲区）的感知。

#### 7. 亮点
*   **仿生设计**：将人类驾驶员的注意力分配逻辑（重点关注+全局余光）转化为算法中的“语义剪枝+多样性回收”，具有很强的直观解释性。
*   **解决 RoPE 偏置**：敏锐察觉到 LLM 原生的位置编码会干扰视觉 Token 的重要性评分，并提出了有效的 RoPE-free 解决方案。
*   **效率与精度双赢**：不仅降低了计算量，还通过过滤无关视觉噪声提升了决策质量。

#### 8. 不足与局限
*   **数据集单一**：实验主要集中在 NAVSIM v2 仿真基准上，尚未在真实世界的复杂路测数据中进行大规模验证。
*   **硬件开销抵消**：虽然减少了 FLOPs，但 ILSA 引入的“Eager Attention”计算和 Token 筛选逻辑在某些硬件架构上可能会带来额外的工程实现开销，实际推理延迟（Latency）的降低可能不如 FLOPs 降幅那么显著。
*   **任务局限**：目前主要针对轨迹预测（Planning），对于更复杂的交互式驾驶任务（如多车博弈）的泛化能力有待进一步观察。

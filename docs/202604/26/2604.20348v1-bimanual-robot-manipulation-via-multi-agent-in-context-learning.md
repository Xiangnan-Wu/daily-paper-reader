## 摘要
语言模型（LLMs）已成为具身控制的强大推理引擎。特别是，上下文学习（ICL）使现成的纯文本 LLM 能够在不进行任何特定任务训练的情况下预测机器人动作，同时保留其泛化能力。将 ICL 应用于双臂操作仍然具有挑战性，因为高维联合动作空间和紧密的手臂间协调约束会迅速超出标准上下文窗口的负荷。为解决这一问题，我们推出了 BiCICLe（双臂协调上下文学习），这是首个使标准 LLM 无需微调即可执行少样本双臂操作的框架。BiCICLe 将双臂控制建模为多智能体主从问题，将动作空间解耦为顺序的、有条件的单臂预测。这自然地扩展到了“手臂辩论”（Arms' Debate）——一个迭代细化过程，并引入了第三个“LLM 作为裁判”来评估并选择最合理的协调轨迹。在 TWIN 基准测试的 13 个任务中进行的评估显示，BiCICLe 实现了高达 71.1% 的平均成功率，比表现最好的免训练基准高出 6.7 个百分点，并超过了大多数监督学习方法。我们进一步展示了在面对新任务时强大的少样本泛化能力。

## Abstract
Language Models (LLMs) have emerged as powerful reasoning engines for embodied control. In particular, In-Context Learning (ICL) enables off-the-shelf, text-only LLMs to predict robot actions without any task-specific training while preserving their generalization capabilities. Applying ICL to bimanual manipulation remains challenging, as the high-dimensional joint action space and tight inter-arm coordination constraints rapidly overwhelm standard context windows. To address this, we introduce BiCICLe (Bimanual Coordinated In-Context Learning), the first framework that enables standard LLMs to perform few-shot bimanual manipulation without fine-tuning. BiCICLe frames bimanual control as a multi-agent leader-follower problem, decoupling the action space into sequential, conditioned single-arm predictions. This naturally extends to Arms' Debate, an iterative refinement process, and to the introduction of a third LLM-as-Judge to evaluate and select the most plausible coordinated trajectories. Evaluated on 13 tasks from the TWIN benchmark, BiCICLe achieves up to 71.1% average success rate, outperforming the best training-free baseline by 6.7 percentage points and surpassing most supervised methods. We further demonstrate strong few-shot generalization on novel tasks.

双臂机器人操作（Bimanual Manipulation）是机器人领域的难点，要求两臂在空间、时间和力学上高度同步。传统的模仿学习（IL）和强化学习（RL）通常需要海量的任务特定数据。
*   **研究动机**：大语言模型（LLM）的上下文学习（ICL）在单臂任务中表现出色，但在双臂任务中面临挑战：
    1.  **高维动作空间**：双臂联合动作空间（$Z^{14}$）会导致动作序列极长，迅速填满 LLM 的上下文窗口。
    2.  **协调复杂性**：简单的动作拼接会导致 LLM 难以捕捉两臂间的精细协作逻辑（如交接、共同抬举）。
*   **核心目标**：提出一种无需训练、仅靠少量示例（Few-shot）即可实现复杂双臂协调操作的框架。

### 2. 方法论：BiCICLe 框架
论文提出了 **BiCICLe**（Bimanual Coordinated In-Context Learning），核心思想是将双臂控制解耦为多智能体协作问题。

*   **主从分解（Leader-Follower Decomposition）**：
    *   **主智能体（Leader）**：首先根据场景观测预测其完整的单臂动作轨迹（$Z^7$）。
    *   **从智能体（Follower）**：在观测的基础上，**以主智能体的计划轨迹为条件**，预测自己的动作。这种分解将 $Z^{14}$ 的预测压力降回 $Z^7$，同时通过显式条件化保证了协调性。
*   **手臂辩论（Arms’ Debate）**：一种迭代优化机制。两臂轮流作为主/从智能体进行二次规划，每一轮都参考对方最新的轨迹进行时空修正。
*   **LLM 裁判（LLM-as-Judge）与 Best-of-N 策略**：生成 $N$ 组候选轨迹，由第三个 LLM 充当裁判，根据演示示例的模式一致性、碰撞风险、抓取逻辑等准则对候选方案评分，选择最优解。
*   **表示方法**：将场景观测（物体质心坐标）和动作（末端执行器位姿+夹持器状态）离散化为文本 Token（如 100x100x100 的体素网格），利用 LLM 的文本补全能力进行推理。

### 6. 主要结论与发现
*   **性能卓越**：BiCICLe + Best-of-N 达到了 **71.1%** 的平均成功率，比最强的免训练基准高出 6.7%，且在多个任务上超越了经过大量数据训练的监督学习模型（如 ACT 和 PerAct 2）。

*   **泛化优势**：在面对全新任务时，BiCICLe 仅需 10 个示例即可上手，而监督学习模型（如 3DFA）在同等数据量下几乎无法工作。

### 7. 优点
*   **架构创新**：首次将多智能体主从博弈引入双臂 ICL，有效解决了高维动作空间导致的上下文溢出问题。
*   **无需训练**：极大地降低了双臂机器人部署的门槛，体现了基础模型“即插即用”的潜力。
*   **鲁棒的细化机制**：通过“辩论”和“裁判”机制，有效抑制了 LLM 采样的随机性和幻觉。

### 8. 不足与局限
*   **精度限制**：依赖于体素化（100³ 分辨率）和离散化的旋转分箱，对于极度精细的操作（如穿针引线、处理细绳）精度可能不足。

*   **推理成本**：Best-of-N 策略虽然提升了性能，但带来了数倍的 Token 消耗和 API 调用成本，实时性较差。
*   **长程任务挑战**：受限于 LLM 的上下文窗口长度，无法在 Prompt 中放入过多的演示示例或处理步骤极多的超长任务。

## 摘要
弥合具身智能与嵌入式部署之间的差距仍然是智能机器人系统中的一个关键挑战，其中感知、推理和规划必须在计算、内存、能量和实时执行的严格约束下运行。在视觉语言导航（VLN）中，现有方法通常在强大的推理能力与真实世界平台上的高效部署之间面临根本性的权衡。在本文中，我们提出了一种可部署的具身 VLN 系统，该系统在真实世界机器人平台上同时实现了高效率和鲁棒的高层推理。为实现这一目标，我们将系统解耦为三个异步模块：用于持续环境感知的实时感知模块、用于空间语义聚合的记忆集成模块，以及用于高层决策的推理模块。我们增量式地构建一个认知记忆图来编码场景信息，并将其进一步分解为子图，以利用视觉语言模型（VLM）进行推理。为了进一步提高导航效率和准确性，我们还利用认知记忆图将探索问题建模为上下文感知的加权旅行维修员问题（WTRP），从而最小化视点的加权等待时间。在仿真和真实世界机器人平台上的广泛实验表明，与现有的 VLN 方法相比，该系统提高了导航成功率和效率，同时在资源受限的硬件上保持了实时性能。

## Abstract
Bridging the gap between embodied intelligence and embedded deployment remains a key challenge in intelligent robotic systems, where perception, reasoning, and planning must operate under strict constraints on computation, memory, energy, and real-time execution. In vision-language navigation (VLN), existing approaches often face a fundamental trade-off between strong reasoning capabilities and efficient deployment on real-world platforms. In this paper, we present a deployable embodied VLN system that achieves both high efficiency and robust high-level reasoning on real-world robotic platforms. To achieve this, we decouple the system into three asynchronous modules: a real-time perception module for continuous environment sensing, a memory integration module for spatial-semantic aggregation, and a reasoning module for high-level decision making. We incrementally construct a cognitive memory graph to encode scene information, which is further decomposed into subgraphs to enable reasoning with a vision-language model (VLM). To further improve navigation efficiency and accuracy, we also leverage the cognitive memory graph to formulate the exploration problem as a context-aware Weighted Traveling Repairman Problem (WTRP), which minimizes the weighted waiting time of viewpoints. Extensive experiments in both simulation and real-world robotic platforms demonstrate improved navigation success and efficiency over existing VLN approaches, while maintaining real-time performance on resource-constrained hardware.

*   **核心问题**：如何在计算资源受限的嵌入式机器人平台上，平衡视觉语言导航（VLN）所需的**高层语义推理能力**与**实时执行效率**。
*   **研究背景**：现有的零样本（Zero-shot）VLN方法通常分为两类：一类是轻量级感知，虽实时但缺乏深度推理；另一类集成大模型（LLM/VLM），推理强但延迟高。此外，现有方法多采用贪婪探索策略，导致在复杂环境中出现往复运动，效率低下。

### 2. 方法论
系统名为 **HiCo-Nav**，核心思想是将导航任务解耦为三个异步运行的分层模块：
*   **分层异步架构**：
    *   **感知-动作层（20Hz）**：负责实时避障、定位和局部规划，确保对环境的即时响应。
    *   **记忆构建层**：增量式构建**认知记忆图（CMG）**。CMG 是一种二分图，包含视觉锚点（关键帧）和对象节点。通过整数线性规划（ILP）优化锚点选择，保持记忆的紧凑性。
    *   **深度推理层（1-2Hz）**：将 CMG 分解为多个子图，异步调用 VLM（如 Qwen3-Omni）进行高层决策，避免阻塞实时控制流。
*   **上下文感知探索与规划**：
    *   **WTRP 建模**：将目标探索问题建模为**加权旅行维修员问题（Weighted Traveling Repairman Problem）**。
    *   **效能评估**：结合“结构语义传播”（利用已知物体推测未知区域）和“边界外视觉证据”（利用图像边缘信息），为候选区域评分。
    *   **全局优化**：通过 WTRP 最小化发现目标的预期时间，而非简单的局部贪婪选择。

### 6. 主要结论与发现
*   **性能领先**：在 MP3D 和 HM3D 上，SR 分别达到 48.5% 和 61.0%，显著优于现有方法。在复杂的 TextNav 任务中，SR 提升了 7.6%。
*   **效率卓越**：任务完成时间比 VLFM 快约 33%，且在嵌入式硬件（Orin NX）上实现了实时运行。
*   **全局规划优势**：WTRP 建模有效减少了冗余路径，提高了长程导航的效率。

### 7. 优点与亮点
*   **异步解耦设计**：巧妙解决了 VLM 推理延迟与机器人实时控制之间的矛盾。
*   **紧凑记忆表示**：通过 ILP 优化的认知记忆图在保留关键语义的同时极大地节省了内存。
*   **全局探索策略**：将探索问题转化为 WTRP，从数学层面优化了“发现时间”，而非仅仅是“路径长度”。
*   **高部署性**：系统不依赖特定环境的预训练，且在资源受限的嵌入式平台上表现稳健。

### 8. 不足与局限
*   **维度限制**：系统目前基于 2D 探索地图，无法处理跨楼层（多层建筑）的导航任务（失败案例中占比约 13.9%-18.5%）。
*   **感知偏差**：在极端视角下存在误检（False Positive），且对小物体（如纸杯、胡萝卜）的识别受机器人运动震动影响较大。
*   **环境动态性**：虽然感知层支持避障，但高层推理对剧烈变动的动态场景处理能力尚待进一步验证。

## 摘要
长程操纵对于视觉-语言-动作（VLA）策略而言仍具挑战性：实际任务通常是多步骤的、依赖进度的，且容易受到复合执行误差的影响。我们提出了 LoHo-Manip，这是一个模块化框架，通过专门的任务管理 VLM 将短程 VLA 执行扩展到长程指令遵循。管理器与执行器解耦，并以滚动时界（receding-horizon）的方式调用：给定当前观测，它预测一个感知进度的剩余计划，该计划结合了：(i) 具有明确“已完成+待完成”划分的子任务序列，作为轻量级语言记忆；(ii) 视觉轨迹（visual trace）——一种紧凑的 2D 关键点轨迹提示，指定下一步的移动路径和接近目标。执行器 VLA 经过调整以根据渲染的轨迹进行条件化，从而通过遵循轨迹将长程决策转化为重复的局部控制。关键在于，在每一步预测剩余计划会产生一个隐式闭环：失败的步骤会保留在后续输出中，轨迹也会相应更新，从而实现自动延续和重新规划，无需手工设计的恢复逻辑或脆弱的视觉历史缓存。在仿真和真实 Franka 机器人上进行的广泛实验（涵盖具身规划、长程推理、轨迹预测和端到端操纵）表明，该方法在长程成功率、鲁棒性和分布外泛化方面取得了显著提升。项目页面：https://www.liuisabella.com/LoHoManip

## Abstract
Long-horizon manipulation remains challenging for vision-language-action (VLA) policies: real tasks are multi-step, progress-dependent, and brittle to compounding execution errors. We present LoHo-Manip, a modular framework that scales short-horizon VLA execution to long-horizon instruction following via a dedicated task-management VLM. The manager is decoupled from the executor and is invoked in a receding-horizon manner: given the current observation, it predicts a progress-aware remaining plan that combines (i) a subtask sequence with an explicit done + remaining split as lightweight language memory, and (ii) a visual trace -- a compact 2D keypoint trajectory prompt specifying where to go and what to approach next. The executor VLA is adapted to condition on the rendered trace, thereby turning long-horizon decision-making into repeated local control by following the trace. Crucially, predicting the remaining plan at each step yields an implicit closed loop: failed steps persist in subsequent outputs, and traces update accordingly, enabling automatic continuation and replanning without hand-crafted recovery logic or brittle visual-history buffers. Extensive experiments spanning embodied planning, long-horizon reasoning, trajectory prediction, and end-to-end manipulation in simulation and on a real Franka robot demonstrate strong gains in long-horizon success, robustness, and out-of-distribution generalization. ### 2. 方法论
核心思想是**分层解耦与闭环规划**：
*   **任务管理器（Task Manager）**：基于预训练 VLM（如 Qwen3-VL），输入当前观测和任务指令，输出：
    1.  **进度感知计划**：以文本形式明确划分“已完成”和“待完成”的子任务序列，作为轻量级记忆。
    2.  **视觉轨迹（Visual Trace）**：在 2D 图像上生成一系列关键点（Waypoints），指明机器人末端执行器应移动的路径。
*   **执行器（Executor）**：一个经过微调的 VLA 模型（如 $\pi_{0.5}$），它以渲染了视觉轨迹的图像为输入，学习“跟随轨迹”来执行具体的短程动作。
*   **关键技术细节**：
    *   **滚动时界规划（Receding-Horizon Planning）**：管理器在每一步（或固定间隔）重新评估环境，如果某步失败，未完成的任务会保留在“剩余计划”中，实现隐式的错误恢复。
    *   **数据流水线**：利用现有的视觉-语言基础模型对原始视频进行自动标注，提取原子子任务和末端执行器轨迹，无需大量人工标注。

### 6. 主要结论与发现
*   **解耦的优越性**：将规划与执行分离，比单体式（Monolithic）VLA 模型在长程任务中更具鲁棒性。
*   **视觉轨迹的有效性**：2D 轨迹作为一种“空间提示”，极大地降低了底层 VLA 的学习难度，使其能更好地泛化到新物体。
*   **隐式恢复机制**：通过每一步预测“剩余计划”，系统无需手动编写复杂的错误检测逻辑即可自动重试失败的步骤。
*   **性能领先**：在 LIBERO 和 VLABench 等多个基准测试中，LoHo-Manip 均取得了 SOTA（当前最佳）性能。

### 7. 优点
*   **模块化强**：任务管理器是执行器无关的，可以轻松更换底层控制模型。
*   **鲁棒性高**：闭环重规划机制能有效应对执行过程中的漂移和意外干扰。
*   **泛化能力**：通过视觉轨迹引导，模型在面对训练集中未见过的物体时表现出极强的零样本（Zero-shot）语义接地能力。
*   **轻量级记忆**：使用文本化的“已完成/剩余”列表代替笨重的视频历史缓存，降低了计算开销。

### 8. 不足与局限
*   **维度限制**：目前的视觉轨迹主要基于 2D 像素坐标，对于需要高度精确的 3D 空间控制或复杂接触力学（如插拔、旋转）的任务可能描述不足。
*   **感知依赖**：系统高度依赖 VLM 的空间接地（Grounding）准确性，如果 VLM 无法准确识别物体位置，生成的轨迹会误导执行器。
*   **场景局限**：实验主要集中在桌面操纵任务，尚未在更动态、更广阔的移动操纵场景中进行验证。

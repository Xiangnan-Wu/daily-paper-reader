## 摘要
在操纵任务中部署生成式世界-动作模型（World-Action Models）正面临冗余像素级重建、$\mathcal{O}(T)$ 内存扩展以及串行推理延迟的严重瓶颈。我们提出了因果潜空间世界模型（CLWM），该模型采用 DINOv3 特征作为生成目标，以将交互语义从视觉噪声中解耦，从而实现极具鲁棒性的领域泛化。为了克服内存扩展问题，CLWM 引入了双状态测试时训练（TTT）内存，确保长程任务中严格的 $\mathcal{O}(1)$ 空间占用。为了降低部署延迟，我们提出了投机异步推理（SAI），通过在物理执行过程中掩盖部分扩散去噪过程，将阻塞延迟降低了约 50%。为了扩展鲁棒策略，我们提出了 EmbodiChain，这是一个在线框架，通过在训练期间注入无限的基于物理的轨迹流来确立效率法则。广泛的实验验证了 CLWM 在复杂双臂仿真中达到了最先进的性能，并在物理机器人上实现了前所未有的零样本 sim-to-real 迁移，其表现优于在真实数据上显式微调的基准模型。

## Abstract
Deploying generative World-Action Models for manipulation is severely bottlenecked by redundant pixel-level reconstruction, $\mathcal{O}(T)$ memory scaling, and sequential inference latency. We introduce the Causal Latent World Model (CLWM), which employs DINOv3 features as generative targets to disentangle interaction semantics from visual noise, yielding highly robust domain generalization. To overcome memory scaling, CLWM features a Dual-State Test-Time Training (TTT) Memory that guarantees a strict $\mathcal{O}(1)$ footprint for long-horizon tasks. To overcome deployment latency, we propose Speculative Asynchronous Inference (SAI) to mask partial diffusion denoising behind physical execution, cutting blocking latency by about $50\%$. To scale robust policies, we present EmbodiChain, an online framework that establishes the Efficiency Law by injecting an infinite flow of physics-grounded trajectories during training. Extensive experiments validate that CLWM achieves state-of-the-art performance in complex dual-arm simulation and unprecedented zero-shot sim-to-real transfer on physical robots, outperforming baselines explicitly finetuned on real-world data.

## 1. 核心问题与研究背景
论文针对当前具身智能领域中**生成式世界-动作模型（World-Action Models, WAMs）**面临的三大瓶颈提出了解决方案：
*   **视觉冗余：** 现有的模型多在像素级或 VAE 潜空间进行预测，导致大量计算资源浪费在与任务无关的背景、光影等细节重建上，难以提取核心交互语义。
*   **内存扩展性差：** 传统的自回归模型依赖 KV Cache，内存占用随任务时长线性增长（$O(T)$），限制了长程任务的处理能力。
*   **推理延迟高：** “感知-计算-执行”的串行模式导致机器人在执行动作时计算单元处于闲置，而在计算时机器人处于停顿，影响闭环控制频率。

## 2. 方法论：核心思想与关键技术
论文提出了 **Causal Latent World Model (CLWM)**，其核心技术包括：
*   **因果潜空间建模：** 放弃像素重建，直接预测 **DINOv3** 提取的高层语义特征。通过将交互语义从视觉噪声中解耦，增强了模型在不同领域（如 Sim-to-Real）的泛化能力。
*   **双状态 TTT 内存（Dual-State TTT Memory）：** 引入**测试时训练（Test-Time Training）**层替代传统的 KV Cache。
    *   **长期内存：** 锚定真实的物理观测，通过梯度下降动态更新 MLP 权重。
    *   **工作内存：** 在推理过程中从长期内存“分叉”，用于处理预测的中间状态，确保内存占用恒定为 $O(1)$。
*   **投机异步推理（Speculative Asynchronous Inference, SAI）：** 利用世界模型的预测能力，在机器人执行当前动作时，利用预测的未来特征提前进行扩散模型的“预去噪”计算。当真实观测到达时，仅需极少量的微调去噪即可输出动作，将阻塞延迟降低了约 50%。
*   **EmbodiChain 框架：** 提出了“具身智能效率法则”，通过在线数据流（Online Data Streaming）持续注入物理真实的仿真轨迹，实现无限的数据多样性。

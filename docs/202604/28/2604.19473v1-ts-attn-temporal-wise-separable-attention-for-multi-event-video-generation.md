## Abstract
Generating high-quality videos from complex temporal descriptions that contain multiple sequential actions is a key unsolved problem. Existing methods are constrained by an inherent trade-off: using multiple short prompts fed sequentially into the model improves action fidelity but compromises temporal consistency, while a single complex prompt preserves consistency at the cost of prompt-following capability. We attribute this problem to two primary causes: 1) temporal misalignment between video content and the prompt, and 2) conflicting attention coupling between motion-related visual objects and their associated text conditions. To address these challenges, we propose a novel, training-free attention mechanism, Temporal-wise Separable Attention (TS-Attn), which dynamically rearranges attention distribution to ensure temporal awareness and global coherence in multi-event scenarios. TS-Attn can be seamlessly integrated into various pre-trained text-to-video models, boosting StoryEval-Bench scores by 33.5% and 16.4% on Wan2.1-T2V-14B and Wan2.2-T2V-A14B with only a 2% increase in inference time. It also supports plug-and-play usage across models for multi-event image-to-video generation. The source code and project page are available at https://github.com/Hong-yu-Zhang/TS-Attn.

这篇论文介绍了一种名为 **TS-Attn**（时间可分离注意力）的创新机制，旨在解决视频生成模型在处理包含多个连续动作（多事件）的复杂描述时面临的挑战。

### 1. 核心问题与整体含义
*   **研究动机**：现有的视频生成模型在处理单事件提示词时表现优异，但在面对“猫先看鱼，再伸爪，最后收回”这类多事件复杂描述时，往往会出现事件遗漏、时序混乱或动作模糊。
*   **核心矛盾**：现有方法存在权衡难题——将复杂提示词拆分为多个短提示词分段生成虽能提高动作保真度，但会导致视频衔接不自然且推理缓慢；而直接输入长提示词虽能保持连贯性，但模型往往无法准确遵循所有指令。
*   **根本原因**：作者通过分析发现，问题根源在于视频内容与文本提示词之间存在**时间错位**，以及运动区域的视频 Token 与多个事件文本条件之间存在**冲突的注意力耦合**。

### 2. 方法论
TS-Attn 是一种**无需训练（Training-free）**的注意力机制，核心思想是：在时间维度上解耦视频与文本的注意力分布，确保运动区域在特定时间段内只聚焦于对应的事件。
*   **运动区域提取 (MRE)**：利用主体（Subject）的语义布局提取跨注意力图，通过自适应阈值和腐蚀操作生成运动区域掩码（Mask），精确定位视频中发生动作的区域。
*   **事件感知注意力调制 (EAM)**：
    *   **注意力重排 (Rearrangement)**：根据预设或生成的事件时间轴，对视频 Token 施加正向或负向偏置（Bias）。在特定时间段内，增强与对应事件词的关联，削弱与其他无关事件词的联系。
    *   **注意力强化 (Reinforcement)**：引入强化因子，根据注意力分布的平滑程度自适应调整缩放比例，确保注意力分布更加尖锐、聚焦。
*   **算法流程**：该机制在推理过程的前 20%-40% 步（去噪早期）替换原始的跨注意力层，通过 LLM（如 GPT-4o-mini）或均匀分割获取事件的时间区间，随后应用上述调制逻辑。

### 3. 实验设计
*   **任务场景**：涵盖文生视频（T2V）和图生视频（I2V）。
*   **基准测试 (Benchmark)**：
    *   **StoryEval-Bench**：包含 423 个提示词，分为人类、动物、物体等 7 个类别，每个提示词含 2-4 个事件。
    *   **StoryEval-Bench-I2V**：作者基于前者构建的图生视频评测集。
*   **对比方法**：
    *   **基础模型**：Wan2.1/2.2, CogVideoX, Open-Sora, Pyramid-Flow 等。
    *   **多事件专用方法**：MEVG, DiTCtrl。
    *   **闭源模型**：KlingAI (可灵), HailuoAI (海螺)。
*   **评估指标**：使用 GPT-4o 和 LLaVA-OV-72B 作为自动评测员，评估事件完整性、时序准确性和主体一致性。

### 4. 资源与算力
*   **硬件**：实验主要在 **NVIDIA A100 GPU** 上进行。
*   **训练时长**：由于是 **Training-free** 方法，该方法**不需要任何训练过程**，直接应用于预训练模型的推理阶段。
*   **推理开销**：相比原始模型，TS-Attn 仅增加了约 **2%** 的推理时间，且 LLM 进行时间规划的平均耗时仅为 2.65 秒。

### 5. 实验数量与充分性
*   **实验规模**：在 4 个不同架构和规模的模型（Wan2.1-1.3B/14B, Wan2.2-A14B, CogVideoX-5B）上进行了验证。
*   **消融实验**：详细测试了 MRE 模块、EAM 模块（重排与强化）以及不同时间分割策略（均匀分割 vs LLM 规划 vs 人工标注）的影响。
*   **充分性评价**：实验设计非常充分，不仅有大规模的定量指标对比，还有丰富的定性视觉结果展示，并扩展到了多主体生成、场景级变换和长视频生成等应用，证明了方法的鲁棒性和公平性。

### 6. 主要结论与发现
*   **性能提升**：在 Wan2.1-14B 上，TS-Attn 将 StoryEval-Bench 的评分提升了 **33.5%**；在 CogVideoX-5B 上提升了 **57.3%**。
*   **时序控制**：通过简单的注意力重排，模型能够严格按照提示词顺序生成动作，有效解决了“事件遗漏”问题。
*   **高效性**：该方法在不破坏全局连贯性的前提下，以极低的计算代价显著增强了模型对复杂指令的遵循能力。

### 7. 优点
*   **即插即用**：无需微调模型，可无缝集成到各种基于 DiT 架构的视频生成模型中。
*   **极低开销**：推理延迟增加微乎其微，远优于需要多次推理的多提示词拼接方法。
*   **解释性强**：基于对注意力图的深入分析提出，针对性地解决了视频生成中的时序耦合冲突。
*   **通用性广**：支持文生视频、图生视频，并能处理多主体和复杂的场景变换。

### 8. 不足与局限
*   **依赖时间规划**：虽然均匀分割也有效，但最佳效果仍依赖于 LLM 或人工提供合理的时间区间划分。
*   **基础模型限制**：生成的视频质量上限仍受限于底座模型（如 Wan 或 CogVideoX）本身的生成能力和最大帧数限制。
*   **掩码精度**：如果提示词中的主体描述过于模糊，MRE 模块提取的运动区域掩码可能不够精确，从而影响调制效果。

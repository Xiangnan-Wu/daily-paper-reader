## 摘要
采用离散标记化方案对动作进行编码的视觉-语言-动作（VLA）模型在机器人操作中得到了越来越多的应用，但现有的解码范式仍存在根本性的局限。无论是通过自回归 VLA 进行序列解码，还是通过离散扩散 VLA 进行并行解码，一旦生成一个标记（token），它通常就是固定的，无法在后续迭代中进行修改，因此早期的标记错误无法在后期得到有效纠正。我们提出了 DFM-VLA，一种用于动作标记迭代细化的离散流匹配 VLA。DFM-VLA 建模了一个标记级的概率速度场，在细化迭代过程中动态更新整个动作序列。我们研究了构建速度场的两种方法：辅助速度头公式化和动作嵌入引导公式化。我们的框架进一步采用了一种两阶段解码策略，包括一个迭代细化阶段，随后是用于稳定收敛的确定性验证。在 CALVIN、LIBERO 和真实世界操作任务上的大量实验表明，DFM-VLA 在保持高推理效率的同时，在操作性能上始终优于强大的自回归、离散扩散和连续扩散基准模型。特别是，DFM-VLA 在 CALVIN 上实现了 4.44 的平均成功长度，在 LIBERO 上实现了 95.7% 的平均成功率，突显了通过离散流匹配进行动作细化对机器人操作的价值。项目主页见 https://chris1220313648.github.io/DFM-VLA/。

## Abstract
Vision--Language--Action (VLA) models that encode actions using a discrete tokenization scheme are increasingly adopted for robotic manipulation, but existing decoding paradigms remain fundamentally limited. Whether actions are decoded sequentially by autoregressive VLAs or in parallel by discrete diffusion VLAs, once a token is generated, it is typically fixed and cannot be revised in subsequent iterations, so early token errors cannot be effectively corrected later. We propose DFM-VLA, a discrete flow matching VLA for iterative refinement of action tokens. DFM-VLA~models a token-level probability velocity field that dynamically updates the full action sequence across refinement iterations. We investigate two ways to construct the velocity field: an auxiliary velocity-head formulation and an action-embedding-guided formulation. Our framework further adopts a two-stage decoding strategy with an iterative refinement stage followed by deterministic validation for stable convergence. Extensive experiments on CALVIN, LIBERO, and real-world manipulation tasks show that DFM-VLA consistently outperforms strong autoregressive, discrete diffusion, and continuous diffusion baselines in manipulation performance while retaining high inference efficiency. In particular, DFM-VLA achieves an average success length of 4.44 on CALVIN and an average success rate of 95.7\% on LIBERO, highlighting the value of action refinement via discrete flow matching for robotic manipulation. #### 1. 核心问题与整体含义（研究动机和背景）
现有的视觉-语言-动作（VLA）模型在处理离散动作 Token 时，主要采用**自回归（AR）**或**离散扩散（DD）**两种范式。然而，这两种方法都存在“**不可逆承诺**”（Irreversible Commitment）问题：一旦某个动作 Token 被生成，它在后续的解码步骤中就无法被修改。这意味着早期的预测错误会随着时间推移而累积，最终导致机器人操作任务失败。
**DFM-VLA** 的研究动机在于引入**离散流匹配（Discrete Flow Matching）**技术，通过建模动作序列的概率速度场，允许模型在迭代过程中对已生成的 Token 进行重新审视和修正，从而提升动作生成的准确性和长程任务的稳定性。

#### 2. 方法论：核心思想与关键技术
DFM-VLA 的核心思想是将动作生成视为一个在离散空间中受速度场引导的连续演化过程。
*   **核心架构**：基于统一的离散 Token 形式，整合文本、视觉（VQ 编码）和动作（FAST 离散化 + BPE 压缩）。
*   **概率速度场构建**：提出了两种变体：
    1.  **辅助速度头（Auxiliary Velocity Head）**：通过额外的预测头从隐藏状态直接预测 Token 替换的速度。
    2.  **动作嵌入引导（Action-Embedding-Guided）**：利用动作 Token 在嵌入空间中的语义距离定义距离度量，构建动力学最优的速度场。实验证明此方法收敛更快、性能更强。
*   **两阶段解码策略**：
    1.  **迭代细化阶段**：利用欧拉离散化采样，根据速度场进行随机跳转和 Token 修正。
    2.  **确定性验证阶段**：在最后几步切换为贪婪解码，以确保轨迹的稳定收敛。
*   **推理优化**：引入**自适应 KV 缓存**策略，根据动作 Token 的变化动态更新缓存，在保持性能的同时实现 2.4 倍的推理加速。

*   **创新性地引入 DFM**：首次将离散流匹配应用于 VLA 领域，解决了离散动作生成的“不可逆”痛点。

*   **两阶段解码**：兼顾了探索性（纠错）和确定性（稳定），设计精巧。

*   **离散化依赖**：性能高度依赖于动作 Tokenizer（如 FAST）的质量，若离散化过程丢失过多精度，流匹配也难以完全弥补。
*   **计算开销**：尽管有加速策略，但相比单次前向传播的自回归模型，多步迭代仍存在一定的计算冗余。
*   **应用限制**：目前主要在机械臂操作任务上验证，对于需要极高频率反馈或极高精度的微操任务，离散 Token 的分辨率可能成为瓶颈。

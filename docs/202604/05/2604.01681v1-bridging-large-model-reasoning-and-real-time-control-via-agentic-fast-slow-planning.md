## 摘要
大型基础模型为自主系统提供了强大的推理能力，但将语义意图映射为可靠的实时控制仍具挑战。现有方法要么 (i) 让大语言模型 (LLM) 直接生成轨迹——这种方式脆弱、难以验证且易产生延迟；要么 (ii) 在线调整模型预测控制 (MPC) 的目标——这混合了慢速深思熟虑与快速控制，模糊了接口界限。我们提出了智能体快慢规划（Agentic Fast-Slow Planning），这是一个在自然时间尺度上解耦感知、推理、规划和控制的分层框架。该框架包含两个桥梁：“感知到决策”（Perception2Decision）利用车载视觉语言模型 (VLM) 检测器将场景压缩为以自我为中心的拓扑结构，随后通过云端 LLM 决策器将其映射为符号化驾驶指令，在保持可解释性的同时减少了带宽和延迟；“决策到轨迹”（Decision2Trajectory）将指令转换为可执行路径，其中“语义引导 A*”（Semantic-Guided A*）将源自语言的软成本嵌入经典搜索中，使解偏向可行轨迹，而“智能体精炼模块”（Agentic Refinement Module）则利用反馈和记忆调整规划器的超参数。最后，MPC 实时跟踪轨迹，并针对困难情况提供可选的云端引导参考。在 CARLA 中的实验表明，与纯 MPC 和 A* 引导的 MPC 基准相比，智能体快慢规划提高了扰动下的鲁棒性，横向偏差减少了高达 45%，完成时间缩短了 12% 以上。代码见 https://github.com/cjychenjiayi/icra2026_AFSP。

## Abstract
Large foundation models enable powerful reasoning for autonomous systems, but mapping semantic intent to reliable real-time control remains challenging. Existing approaches either (i) let Large Language Models (LLMs) generate trajectories directly - brittle, hard to verify, and latency-prone - or (ii) adjust Model Predictive Control (MPC) objectives online - mixing slow deliberation with fast control and blurring interfaces. We propose Agentic Fast-Slow Planning, a hierarchical framework that decouples perception, reasoning, planning, and control across natural timescales. The framework contains two bridges. Perception2Decision compresses scenes into ego-centric topologies using an on-vehicle Vision-Language Model (VLM) detector, then maps them to symbolic driving directives in the cloud with an LLM decision maker - reducing bandwidth and delay while preserving interpretability. Decision2Trajectory converts directives into executable paths: Semantic-Guided A* embeds language-derived soft costs into classical search to bias solutions toward feasible trajectories, while an Agentic Refinement Module adapts planner hyperparameters using feedback and memory. Finally, MPC tracks the trajectories in real time, with optional cloud-guided references for difficult cases. Experiments in CARLA show that Agentic Fast-Slow Planning improves robustness under perturbations, reducing lateral deviation by up to 45% and completion time by over 12% compared to pure MPC and an A*-guided MPC baseline. Code is available at https://github.com/cjychenjiayi/icra2026_AFSP.

### 2. 方法论：核心思想与关键技术
AFSP 框架通过两个关键“桥梁”连接不同层级：

*   **Perception2Decision（感知到决策）**：
    *   **车载端（快）**：使用轻量化 VLM（视觉语言模型）作为拓扑检测器，将原始图像压缩为以自我为中心的“拓扑图”（包含物体类别、距离、方位等），极大降低了上传云端的带宽需求。
    *   **云端（慢）**：LLM 接收拓扑图，通过思维链（CoT）推理生成符号化的驾驶指令（如 LEFT, KEEP, RIGHT）及驾驶风格。
*   **Decision2Trajectory（决策到轨迹）**：
    *   **语义引导 A\* 算法**：这是本文的核心创新。作者将 LLM 的语义指令转化为 A\* 搜索中的“软成本”（Soft Cost）。如果路径符合指令则给予奖励，不符合则惩罚，从而在保证几何可行性的前提下，引导搜索趋向语义意图。
    *   **智能体精炼模块（Agentic Refinement）**：利用 LLM 作为一个闭环调节器，根据规划反馈（如是否碰撞、是否偏离意图）自动调整 A\* 的超参数，替代了繁琐的人工调参。
*   **实时控制层**：使用 **云端引导的 MPC**。在常规情况下由本地 MPC 跟踪轨迹，在复杂或困难场景下参考云端生成的全局路径，确保实时响应与安全性。

### 3. 实验设计
*   **实验环境**：使用 **CARLA** 自动驾驶仿真器。
*   **数据集**：构建了一个包含 49 类物体、约 20 万帧图像的自定义数据集（涵盖 Town2-7 和 Town10HD），用于训练 VLM 拓扑检测器。
*   **对比基准（Benchmarks）**：

### 6. 主要结论与发现
*   **性能提升**：AFSP 在所有场景中均表现最优。相比纯 MPC，**横向偏差减少了约 45%**，任务**完成时间缩短了 12% 以上**。
*   **鲁棒性**：在地图存在 1-2 米偏移的扰动情况下，传统 A\* 容易失效或产生震荡，而语义引导的 A\* 能够凭借高层意图保持路径的连贯性。

### 7. 优点与亮点
*   **解耦设计**：成功将“慢速语义推理”与“快速物理控制”分离，解决了大模型直接控制时的实时性难题。
*   **软成本接口**：提出了一种将自然语言指令嵌入经典图搜索算法（A\*）的原则性方法，这比硬约束更具容错性。
*   **自动调参**：引入 Agentic Refinement 模块，利用 LLM 的反思能力自动优化规划器参数，显著降低了部署成本。

### 8. 不足与局限
*   **云端依赖**：系统高度依赖云端 LLM 的决策。如果发生网络中断或极端高延迟，虽然有本地 MPC 保底，但高层智能将丧失。

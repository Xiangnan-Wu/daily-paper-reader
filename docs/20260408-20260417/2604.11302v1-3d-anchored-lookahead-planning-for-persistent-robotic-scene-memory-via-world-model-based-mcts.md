## 摘要
我们提出了 3D 锚定前瞻规划（3D-ALP），这是一种用于机器人操作的系统 2 推理引擎，它将蒙特卡洛树搜索（MCTS）与作为展开预言机（rollout oracle）的 3D 一致性世界模型相结合。与仅根据当前相机帧评估动作的反应式策略不同，3D-ALP 维护一个持久的相机到世界（c2w）锚点，该锚点在遮挡情况下依然保持，从而能够对不再直接可见的物体位置进行准确的重新规划。在需要空间记忆的 5 步序列到达任务（实验 E3）中，3D-ALP 在需要记忆的步骤上实现了 0.650 ± 0.109 的成功率，而贪婪反应式基准仅为 0.006 ± 0.008（Δ=+0.645）；在第 5 步的成功率上，3D-ALP 达到了 0.822，而贪婪基准为 0.000。消融实验（30 个回合，3 个随机种子）表明，树搜索空间记忆是主要的驱动因素（+0.533，占增益的 82%），更深的前瞻规划也带来了额外收益（+0.111，占 17%）。我们还识别并解决了将 UCT-MCTS（应用于树的上置信界算法）应用于连续机器人操作时的四种结构性失效模式。

## Abstract
We present 3D-Anchored Lookahead Planning (3D-ALP), a System 2 reasoning engine for robotic manipulation that combines Monte Carlo Tree Search (MCTS) with a 3D-consistent world model as the rollout oracle. Unlike reactive policies that evaluate actions from the current camera frame only, 3D-ALP maintains a persistent camera-to-world (c2w) anchor that survives occlusion, enabling accurate replanning to object positions that are no longer directly observable. On a 5-step sequential reach task requiring spatial memory (Experiment E3), 3D-ALP achieves 0.650 0.109 success rate on memory-required steps versus 0.006 0.008 for a greedy reactive baseline (Δ=+0.645), while step 5 success reaches 0.822 against 0.000 for greedy. An ablation study (30 episodes, 3 seeds) isolates tree search spatial memory as the primary driver (+0.533, 82% of gain) with additional benefit from deeper lookahead (+0.111, 17%). We also identify and resolve four structural failure modes in applying UCT-MCTS (Upper Confidence Bounds applied to Trees [10]) to continuous robotic manipulation.

### 1. 核心问题与整体含义（研究动机和背景）
现代机器人操作主要依赖“系统 1”（System 1）式的反应式策略（如 VLA 模型），它们直接将当前视觉帧映射为动作。然而，这类模型缺乏**物体恒常性（Object Permanence）**，即当物体被遮挡或移出视野后，机器人会“忘记”物体的位置。论文提出，机器人需要一种“系统 2”（System 2）推理引擎，通过维护持久的 3D 空间记忆和前瞻性规划（Lookahead Planning），在物体不可见时仍能准确执行任务。

### 2. 方法论
3D-ALP 的核心思想是将**蒙特卡洛树搜索（MCTS）**与**3D 一致性世界模型**相结合。
*   **持久 3D 锚点（c2w）**：系统维护一个相机到世界坐标系的变换矩阵。每当机器人移动，锚点通过正向运动学（FK）更新而非重置，确保即使物体被遮挡，其在 3D 空间中的相对位置信息仍保留在规划树中。
*   **世界模型预言机**：使用 `InSpatio-WorldFM` 作为模拟器，根据 MCTS 采样的候选动作渲染预测的未来视觉帧。
*   **混合几何-语义评分器**：为了解决视觉大模型（VLM）在深度感知上的“盲区”（即 2D 投影重合但 3D 距离尚远），论文引入了混合评分公式：总分 = 语义得分（来自 Florence-2）× 运动学深度惩罚项。
*   **MCTS 的四项结构性修复**：
    1.  **F1（零动作陷阱）**：通过 Max-Q 值选择并过滤零增量动作，防止机器人原地打转。
    2.  **F2（树深度衰减）**：在根节点移动后递归重置深度，保持前瞻视界。
    3.  **F3（标准平均惩罚）**：采用 Max-MCTS，反向传播最大值而非平均值，避免优秀路径被差路径稀释。
    4.  **F4（UCB1 常数失配）**：将探索常数 $c$ 从 1.414 调低至 0.02，以匹配连续空间中微小的运动学得分差异。

### 3. 实验设计
*   **场景与任务**：在 MuJoCo 仿真环境中使用 Franka Panda 机械臂执行 5 步序列触达任务（Experiment E3）。前 3 步目标可见，第 4-5 步要求机器人返回之前访问过但当前已不在视野内的位置。
*   **基准测试（Benchmark）**：对比了**贪婪反应式基准（Greedy reactive baseline）**，该基准在每一步仅根据当前帧选择即时距离最小的动作。
*   **对比方法**：

    *   MCTS D=1（1 层深度搜索，验证空间记忆贡献）。
    *   3D-ALP D=2（2 层深度搜索，验证前瞻规划贡献）。

### 5. 实验数量与充分性
*   **实验规模**：主实验 E3 包含 **30 个回合 × 3 个随机种子**，共 90 次完整测试。
*   **消融实验**：详细分解了“树搜索记忆”与“搜索深度”的贡献占比。
*   **充分性评价**：实验设计针对性极强，通过对比可见步骤与不可见步骤，清晰地隔离了空间记忆的变量。统计学上，3D-ALP 与基线的差异达到了显著性水平（p < 0.001）。

### 6. 主要结论与发现
*   **记忆鸿沟**：在需要记忆的步骤中，反应式基线成功率几乎为零（0.6%），而 3D-ALP 维持在 **65%**。在最难的第 5 步，3D-ALP 成功率达到 **82.2%**。
*   **增益分解**：消融研究显示，**82% 的性能提升源于树搜索提供的空间记忆**，另外 17% 源于更深的前瞻规划（D=2）。
*   **VLM 局限性**：实验证实当前 VLM 无法提供高精度的 3D 奖励信号，必须结合几何约束。

### 7. 优点
*   **解决核心痛点**：显式地为机器人提供了“物体恒常性”，这是迈向复杂长程操作的关键。
*   **模块化与可组合性**：3D-ALP 的架构允许独立更换世界模型、评分器或 MCTS 引擎，具有很强的通用性。
*   **算法鲁棒性**：针对连续控制环境对经典 MCTS 进行了必要的数学修正。

### 8. 不足与局限
*   **渲染瓶颈**：世界模型的渲染速度（2.4秒/帧）限制了实时规划的吞吐量，目前难以直接应用于高速动态环境。
*   **感知偏差**：高度依赖精确的正向运动学（FK）和相机外参，若物理锚点发生漂移，规划将失效。
*   **环境限制**：目前仅在 MuJoCo 仿真中验证，尚未在复杂的真实世界物理环境（如光照变化、非刚体物体）中进行大规模真机测试。

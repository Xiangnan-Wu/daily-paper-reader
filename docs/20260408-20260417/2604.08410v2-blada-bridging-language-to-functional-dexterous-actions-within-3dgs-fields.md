## 摘要
在非结构化环境中，功能性灵巧抓取需要语义理解、精确的 3D 功能定位以及物理可解释执行的紧密集成。模块化分层方法比端到端 VLA（视觉-语言-动作）方法更具可控性和可解释性，但现有方法仍依赖于预定义的示能（affordance）标签，且缺乏功能性灵巧操控所需的紧密语义-姿态耦合。为了解决这一问题，我们提出了 BLaDA（Bridging Language to Dexterous Actions in 3DGS fields），这是一个可解释的零样本框架，它将开放词汇指令落地为功能性灵巧操控的感知和控制约束。BLaDA 通过知识引导语言解析（KLP）模块首先将自然语言解析为操控约束的结构化六元组，从而建立起一条可解释的推理链。为了实现姿态一致的空间推理，我们引入了三角功能点定位（TriLocation）模块，该模块利用 3D 高斯泼溅（3DGS）作为连续场景表示，并在三角几何约束下识别功能区域。最后，3D 关键点抓取矩阵变换执行（KGT3D+）模块将这些语义-几何约束解码为物理上合理的腕部姿态和手指级指令。在复杂基准测试上的大量实验表明，BLaDA 在示能落地精度以及跨不同类别和任务的功能性操控成功率方面均显著优于现有方法。代码将在 https://github.com/PopeyePxx/BLaDA 公开。

## Abstract
In unstructured environments, functional dexterous grasping calls for the tight integration of semantic understanding, precise 3D functional localization, and physically interpretable execution. Modular hierarchical methods are more controllable and interpretable than end-to-end VLA approaches, but existing ones still rely on predefined affordance labels and lack the tight semantic--pose coupling needed for functional dexterous manipulation. To address this, we propose BLaDA (Bridging Language to Dexterous Actions in 3DGS fields), an interpretable zero-shot framework that grounds open-vocabulary instructions as perceptual and control constraints for functional dexterous manipulation. BLaDA establishes an interpretable reasoning chain by first parsing natural language into a structured sextuple of manipulation constraints via a Knowledge-guided Language Parsing (KLP) module. To achieve pose-consistent spatial reasoning, we introduce the Triangular Functional Point Localization (TriLocation) module, which utilizes 3D Gaussian Splatting as a continuous scene representation and identifies functional regions under triangular geometric constraints. Finally, the 3D Keypoint Grasp Matrix Transformation Execution (KGT3D+) module decodes these semantic-geometric constraints into physically plausible wrist poses and finger-level commands. Extensive experiments on complex benchmarks demonstrate that BLaDA significantly outperforms existing methods in both affordance grounding precision and the success rate of functional manipulation across diverse categories and tasks. Code will be publicly available at https://github.com/PopeyePxx/BLaDA.

灵巧手是人形机器人的核心末端执行器，但在非结构化环境中实现“功能性抓取”（如：不仅是拿起电钻，还要按下开关使用它）极具挑战。
*   **研究动机**：现有的端到端视觉-语言-动作（VLA）模型虽然强大，但往往是“黑盒”且依赖海量数据，泛化性差；而传统的模块化方法多局限于简单的取放任务，缺乏语义理解与精确 3D 姿态控制的紧密耦合。
*   **核心问题**：如何构建一个可解释的、零样本（Zero-shot）的框架，将开放词汇的自然语言指令直接转化为物理上精确且可执行的灵巧手操控动作？

#### 2. 方法论：核心思想与技术细节
BLaDA 提出了一个三阶段的推理链条，将抽象指令逐步落地为物理控制：
*   **知识引导语言解析 (KLP)**：
    *   利用大语言模型（LLM）结合结构化知识图谱，将自然语言指令解析为**结构化六元组** $S = (g_a, g_r, g_t, g_f, \tau, \kappa)$。
    *   包含：可用区域（$g_a$）、手指角色分配（$g_r$）、抓取类型（$g_t$）、交互力（$g_f$）、工具拓扑（$\tau$）和任务意图（$\kappa$）。
*   **三角功能点定位 (TriLocation)**：
    *   **3DGS 场景表示**：使用 3D 高斯泼溅构建连续的 3D 语义场。
    *   **分层语义提取 (HSE)**：通过解耦物体级和零件级特征，解决语义漂移问题，精准识别功能部件（如喷壶的按钮）。
    *   **几何约束定位**：在 3D 空间中确定三个关键点（功能点 $p_1$、侧向支撑点 $p_2$、腕部支撑点 $p_3$），形成稳定的三角几何结构。
*   **3D 关键点抓取矩阵变换执行 (KGT3D+)**：
    *   将定位的 3D 关键点解码为机器人的腕部姿态（旋转 $R$ 和平移 $T$）。
    *   结合解析出的抓取类型和力参数，生成手指级的关节指令和力控制方案。

#### 6. 主要结论与发现
*   **KLP 的有效性**：引入知识引导后，LLM 的推理准确率平均提升了约 **21.5%**，尤其在涉及特定手指功能分配时表现显著。
*   **定位精度**：TriLocation 模块在复杂场景下的定位成功率（LSR）达到 **68.75%**，远超基准方法 MKA（22.5%）。
*   **执行成功率**：在真实世界任务中（如“按下喷壶”），BLaDA 的成功率显著高于端到端策略，证明了显式语义-几何约束在零样本任务中的优越性。

#### 7. 优点（亮点）
*   **零样本迁移**：无需针对特定任务进行大规模策略训练，即可处理开放词汇指令。
*   **物理可解释性**：通过结构化六元组和 3D 关键点，使机器人从“理解指令”到“执行动作”的每一步都清晰可见。
*   **3DGS 集成**：巧妙利用 3DGS 的连续性，实现了比点云或 2D 图像更精准的细粒度零件定位。

#### 8. 不足与局限
*   **语义模糊性**：现有的视觉语言模型在解析工具部件时仍可能出现歧义（例如将工具的“头部”误认为人的头部）。
*   **缺乏触觉反馈**：系统目前主要依赖视觉，若物体在抓取过程中发生意外位移或滑动，缺乏闭环的动态调整能力。
*   **零件特征稀疏**：在处理极小或特征不明显的零件时，3DGS 的语义场可能不够密集，导致定位失败。

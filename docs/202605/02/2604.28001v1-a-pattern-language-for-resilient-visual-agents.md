## 摘要
将多模态基础模型集成到企业生态系统中提出了一个根本性的软件架构挑战。架构师必须平衡相互竞争的质量属性：视觉语言动作（VLA）模型的高延迟和非确定性，与企业控制回路所需的严格确定性和实时性能。在本研究中，我们为视觉智能体提出了一种架构模式语言，该语言将快速、确定性的反射与慢速、概率性的监督分离开来。它由四种架构设计模式组成：(1) 混合示能性集成，(2) 自适应视觉锚定，(3) 视觉层级综合，以及 (4) 语义场景图。

## Abstract
Integrating multimodal foundation models into enterprise ecosystems presents a fundamental software architecture challenge. Architects must balance competing quality attributes: the high latency and non-determinism of vision language action (VLA) models versus the strict determinism and real-time performance required by enterprise control loops. In this study, we propose an architectural pattern language for visual agents that separates fast, deterministic reflexes from slow, probabilistic supervision. It consists of four architectural design patterns: (1) Hybrid Affordance Integration, (2) Adaptive Visual Anchoring, (3) Visual Hierarchy Synthesis, and (4) Semantic Scene Graph.

### 2. 方法论：核心思想与关键技术
论文提出了一个包含三个异步控制回路（反射、结构、监督）的分层参考架构，并定义了四种核心设计模式：
*   **混合示能性集成（Hybrid Affordance Integration）**：
    *   **核心思想**：多模态感知融合。并行运行轻量级本地模型（如 UI 检测器和 OCR），通过几何融合和冲突解决机制，过滤感知噪声，避免“感知幻觉”。
*   **自适应视觉锚定（Adaptive Visual Anchoring）**：
    *   **核心思想**：建立“系统 1”本地缓存与“系统 2”VLA 监督者之间的回退路由。
    *   **技术细节**：正常情况下使用 <50ms 的本地模板匹配；当置信度低于阈值（如 $\tau < 0.9$）时，触发高延迟的 VLA 进行语义定位并更新缓存，实现**摊销推理（Amortized Inference）**。
*   **视觉层级综合（Visual Hierarchy Synthesis）**：
    *   **核心思想**：利用格式塔原理（如邻近性、对齐性）将扁平的检测结果转化为层级化的 UI 树，解决绝对坐标失效导致的歧义问题。
*   **语义场景图（Semantic Scene Graph）**：
    *   **核心思想**：构建一个显式的、可查询的世界模型。在执行高风险操作前，通过遍历场景图验证前提条件，增强系统的可解释性和安全性。

### 3. 实验设计
*   **评估方法**：采用了**基于场景的架构分析方法（SAAM）**。
*   **实验场景**：模拟了一个典型的金融 ERP 系统中的“发票审批”工作流。
*   **压力事件（UI 漂移）**：供应商更新了 UI，导致“提交”按钮移动了位置、改变了颜色，并在原位置出现了一个具有破坏性的“删除”按钮。
*   **对比基准（Benchmarks）**：
